# School Management Frontend

A production-ready React frontend for the school management system.

## Getting Started

1. Install dependencies:
   ```
   npm install
   ```

2. Start the dev server:
   ```
   npm run dev
   ```

3. Build for production:
   ```
   npm run build
   ```

## Environment

Create a `.env` file:
```
VITE_API_URL=http://localhost:5000/api
```

## Features

- Student Dashboard: View attendance, marks, and activity
- Teacher Dashboard: Manage students, attendance, and marks
- Admin Dashboard: System overview and statistics
- Role-based access control
- Responsive UI with sidebar navigation

## Technology Stack

- React 18
- React Router DOM v6
- Axios
- Vite
- Context API for state management
